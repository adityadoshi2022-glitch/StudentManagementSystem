# Student Management System

## Overview
A simple Java console-based application to manage students.
Allows adding, viewing, searching, and deleting student records.

## Features
- Add student
- View all students
- Search student by ID
- Delete student by ID

## Tech Stack
- Java
- ArrayList for data storage
- Console-based interface

## How to Run
1. Open terminal/command prompt
2. Navigate to src folder: `cd Desktop\StudentManagementSystem\src`
3. Compile all files: `javac *.java`
4. Run: `java Main`
