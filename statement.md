\# Student Management System



\## Problem Statement

Educational institutions need a simple system to store and manage basic student information. 

Currently, many schools/colleges maintain records manually, which is time-consuming and error-prone. 

This system allows adding, viewing, searching, and deleting students efficiently.



\## Scope of the Project

\- Manage student information digitally

\- Add new student records

\- View all students at a glance

\- Search and delete students by ID

\- Suitable for small schools, colleges, or training institutes



\## Target Users

\- School/college administrators

\- Teachers

\- Training institute staff



\## High-Level Features

1\. Add student (ID, name, age)  

2\. View all students  

3\. Search student by ID  

4\. Delete student by ID  

5\. Console-based Java application (no GUI required)
